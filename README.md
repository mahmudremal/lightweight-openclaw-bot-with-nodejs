### Romi - AI Personal Assistant

Build a Node.js application named **Romi**, an AI assistant inspired by the OpenClaw architecture. Refer to the `nanobot-main` repository for a similar implementation built in Python to ensure functional parity. You can look at ther how they made it. right?

**Core Requirements:**

- **LLM Provider:** Integrate OpenAI API as the sole provider.
- **Channel:** Implement WhatsApp integration using a headless solution like `whatsapp-web.js`.
- **Search Tool:** Use DuckDuckGo for web search capabilities.
- **Skills:** Implement a weather skill.
- **Automation:** Include cron jobs and background tasks consistent with the OpenClaw framework.
- **Architecture:** Ensure the application is lightweight, modular, and follows the logic of the provided Python reference while being native to Node.js.

# Skills Usage

You have access to a variety of skills that extend your capabilities.

## How to use skills:

1. Check the **Available skills** list below.
2. If a task relates to any of these skills, you **MUST** first read the skill's documentation using `read_file('skills/[skill-name]/SKILL.md')`. Cause on every skills, there is a file `SKILL.md` which you need to read on necessity.
3. Follow the instructions and patterns documented in the skill to perform the task effectively.

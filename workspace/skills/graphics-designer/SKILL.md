# Graphics Designer Skill

I handle visual content creation by automating the browser.

## Workflow

To generate images or music, use the `terminal_exec` tool (or `spawn_subagent`) to run the `nanobanana` script.

### 🖼️ Generate Image

`node scripts/nanobanana.js image "A futuristic city with neon lights"`

### 🎵 Generate Music

`node scripts/nanobanana.js music "Melodic lofi hip-hop beat"`

## Capabilities

- **Image Generation**: Generates high-quality photorealistic or artistic images.
- **Music Generation**: Creates ambient or rhythmic audio tracks.
- **Automation**: One-command process including navigation, mode selection, and content extraction.

> [!NOTE]
> Ensure the Romi Browser extension is connected before running the script.
